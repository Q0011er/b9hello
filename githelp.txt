Настройка гита:

git config --global user.name "My Name"
git config --global user.email myEmail@example.com

идем в директорию с проектом и создаем там репозитарий:
$ git init

смотрим статус репозитария
$ git status

выставляем всем файлам статус отслеживаемых (кроме указанных в .gitignore)

$ git add -A (либо --all)

коммитим изменения (в первый раз тупо запихиваем все отселживаемые файлы
в последсвии в репозитарий будут записаны только изменения)

$ git commit -m "Первый коммит - Поехали!!!"

создаем ссылку на удаленный сервер:
$ git remote add origin  https://github.com/myAccount/awesome-project.git

// origin - это сокращенное название удаленного репозитария, чтобы в последующих командах использовать его а не длинный URL, поэтому вы можете назвать его как хотите (по стандарту пишут origin, но я люблю придумывать имена, чтобы не путаться).

репозитарий может находится в разных ветках и даже на разных серверах (github, gitlab, bitbucket и т.д.) чтобы посмотреть все сслки на удаленные репозитарии набираем:

$ git remote -v

чтобы залить репозитарий на удаленный сервер, воспользуемся созданной выше ссылкой на него:

$ git push origin master

// master - это наша основная ветка в репозитарии
